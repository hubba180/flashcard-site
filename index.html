JavaScript Flashcards


  • JavaScript (JS)
  • often abbreviated as JS, is a programming language that conforms to the ECMAScript specification
  • Operators
  • constructs which behave generally like functions, but which differ syntactically or semantically from usual functions.
  • Variables
  • a variable or scalar is a storage address paired with an associated symbolic name, which contains some known or unknown quantity of information referred to as a value.
  • Variable naming conventions
  • camelCase
  • Functions
  • a named section of a program that performs a specific task.
  • Methods
  • a procedure associated with a class.
  • Arguments
  • a value that is passed between programs, subroutines or functions.
  • Parameters
  • a special kind of variable in that is used to pass information between functions or procedures.
  • Return
  • a return statement causes execution to leave the current subroutine and resume at the point in the code immediately after the instruction which called the subroutine, known as its return address
  • Chaining methods
  • also known as named parameter idiom, is a common syntax for invoking multiple method calls in object-oriented programming languages.
  • Strings
  • set of characters surrounded by ""
  • Booleans
  • data type that has one of two possible values which is intended to represent the two truth values of logic and Boolean algebra.
  • Undefined
  • a condition where an expression does not have a correct value, although it is syntactically correct
  • NaN
  • Not a Number: an undefined or unrepresentable value
  • Null
  • a built-in constant that has a value of zero
  • Alert
  • function that brings up a pop up alert window
  • Comments
  • ignored code
  • jQuery
  • javascript library
  • Attributes
  • a specification that defines a property of an object, element, or file.
